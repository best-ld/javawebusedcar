# JAVA二手车交易信息平台

## 一、要求

#### 1、会员

会员注册，基本信息修改；二手汽车出售信息的编辑、修改、发布（发布后不可再修改）、关闭（关闭后他人不可再留言）；查询并维护自己的出售信息，查询他人发布的出售信息并可以留言，可以对他人的留言进行回复；

#### 2、管理员

会员管理（权限管理，例如是否可以发布出售信息，取消用户会员资格等），出售信息管理（管理员拥有对虚假消息进行管理的权力，可以屏蔽虚假消息）。
